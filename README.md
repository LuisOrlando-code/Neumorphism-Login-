# Neumorphic Login & Registration Form

Un formulario de inicio de sesión y registro moderno con diseño **Neumórfico** (Soft UI). Este proyecto destaca por una transición fluida y animaciones elegantes entre los estados de "Sign In" y "Sign Up" utilizando HTML, CSS y JavaScript puro.

## 🚀 Características

*   **Diseño Neumórfico:** Uso de sombras suaves (`box-shadow`) para crear un efecto de relieve y profundidad.
*   **Animación de Intercambio:** Transición lateral suave entre los formularios de registro y login.
*   **Totalmente Interactivo:** Botones con estados *hover* y animaciones de "estiramiento" durante el cambio de panel.
*   **Diseño Responsivo:** Adaptado para diferentes resoluciones de pantalla mediante el uso de `transform: scale()`.

## 🛠️ Tecnologías utilizadas

*   **HTML5:** Estructura semántica.
*   **CSS3:** Flexbox, animaciones, transiciones y variables para el diseño neumórfico.
*   **JavaScript (Vanilla):** Lógica de manipulación del DOM para el intercambio de clases y control de tiempos de animación.
*   **Google Fonts:** Fuente 'Ubuntu' para una tipografía limpia.
